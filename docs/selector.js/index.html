UI Lab, Selector JS

Selector JS is a simple javascript selector to select HTML elements.

Please, click and check the console to see returned element/array/nodelist. Selector JS Github
  • Select tagname: selector('li');
  • Select multiple tagname: selector('li, i');
  • Select first tagname: selector('li')[0];
  • Select CSS3 last-child of tagname: selector('li:last-child');
  • Select Classname: selector('.select-classname');
  • Select id: selector('#select-id');
  • Select with attribute: selector('[onclick]');
  • Select document: selector(document);
  • Select window: selector(window);
  • Select this element: selector(this);
  • Find tagname in this element: selector('i', this);
  • Find classname in selected classname: selector('.inner-classname', '.outer-classname');
  • Find id in selected classname: selector('#inner-id', '.outer-classname2');
