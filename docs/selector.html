UI Lab, Selector

Selector JS is a simple javascript selector to select HTML elements.
Select element: 
selector('element');
Select multiple elements: 
selector('element, element');
Select first of elements: 
selector('element')[0];
Select CSS3 last-child of elements: 
selector(':last-child');
Select classname: 
selector('.class');
Select ID: 
selector('#id');
Select with attribute: 
selector('[attribute]');
Select document: 
selector(document);
Select window: 
selector(window);
Select this element: 
selector(this);
Find an element in this element: 
selector('element', this);
Find classname in parent classname: 
selector('.class', '.parentClass');
Find ID in selected classname:: 
selector('#id', '.class');
