UI Lab, Alerts

Showing Dialogues

alerts.dialog({ msg: '', success: '', error: '', custom: { value: 'button text', value: 'button text' }, callback });
msg
Required. Your dialog text.
success
Optional. Set success button text.
Success button returns a value: 'success'
error
Optional. Set error button text.
Error button returns a value: 'error'
custom
Optional. Set custom buttons.
Custom buttons returns defined own value.
Example usage: {value: 'button text', value: 'button text'}
callback
Optional. Set a callback function when dialog returns a value.
Example usage: function (value) {}
Examples: 
alerts.dialog({
    msg: 'Your dialog text'
});
alerts.dialog({
    msg: 'Your dialog text',
    success: 'Got it!'
});
alerts.dialog({
    msg: 'Your dialog text',
    success: 'Got it!',
    error: 'No, thanks!'
});
alerts.dialog({
    msg: 'Your dialog text',
    success: 'Got it!',
    error: 'No, thanks!',
    custom: {
        maybe: 'Maybe', later: 'Later'
    }
});
alerts.dialog({
    msg: 'Your dialog text',
    success: 'Got it!',
    error: 'No, thanks!',
    custom: {
        maybe: 'Maybe', later: 'Later'
    },
    callback: function (value) {
        alert(value);
    }
});
Submit form example: 
return alerts.dialog({
    msg: 'Submit this form?',
    success: 'Yes',
    error: 'No',
    callback: function (value) {
        if (value === 'success') {
            document.getElementById('submitForm').submit();
        }
    }
});

Showing Messages

alerts.message({ msg: '', pos: '', theme: '' });
msg
Required. Your alert message.
pos
Optional. Set message position.
Default value: 'tr'
Available values: 'tr', 'tl', 'br', 'bl'
theme
Optional. Set message theme.
Available values: 'success', 'warning', 'danger'
Show Positions: 
alerts.message({
    msg: 'Top left message',
    pos: 'tl'
});
alerts.message({
    msg: 'Bottom left message',
    pos: 'bl'
});
alerts.message({
    msg: 'Top right message',
    pos: 'tr'
});
alerts.message({
    msg: 'Bottom right message',
    pos: 'br'
});
Show Default:
Default message 
alerts.message({
    msg: 'Default message'
});
Show Success:
Success message 
alerts.message({
    msg: 'Success message',
    theme: 'success'
});
Show Warning:
Warning message 
alerts.message({
    msg:'Warning message',
    theme: 'warning'
});
Show Danger:
Danger message 
alerts.message({
    msg: 'Danger message',
    theme: 'danger'
});
