Events JS Examples

Events JS requires Selector JS.

Events JS is a simple javascript creating and triggering events. Please, click and check the event results.

Add events:

  • Add click event: events.on('.btn-click-add', 'click', function () { alert('Button click.'); });
  • Add mousedown event: events.on('.btn-mousedown-add', 'mousedown', function () { alert('Button mousedown.'); });

Remove events:

  • Remove click event: events.off('.btn-click-remove', 'click');
  • Remove mousedown event: events.off('.btn-mousedown-remove', 'mousedown');

Trigger events:

  • Trigger click event: events.trigger('.btn-click-trigger', 'click');
  • Trigger mousedown event: events.trigger('.btn-mousedown-trgigger', 'mousedown');

Check classname:

  • Check element has a class named "new-class": events.hasClass('.btn-click-check', 'new-class');

Add/Remove classname:

  • Add a class name "new-class": events.addClass('.btn-class-add', 'new-class');
  • Remove a class named "old-class": events.removeClass('.btn-class-remove', 'old-class');

Show/Hide element:

  • Show element "<i>": events.show('.btn-show i');
  • Hide element "<i>": events.hide('.btn-hide i');

.each() Run for each matched element:

  • Check the console to see the results: events.each('.btn-each b', function () { console.log(this.innerHTML); }); First matched element Second matched element Third matched element

.closest() Returns the first ancestor:

  • Check the console to see the results: events.closest('.btn-closest', 'div.parent');

* Compatible with all modern browsers and IE9+
