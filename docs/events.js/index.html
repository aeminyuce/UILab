UI Lab, Events JS

Events JS is a simple javascript creating and triggering events.

Events JS requires Selector JS.
Please, click the buttons listed below and check the event results. Events JS Github

Add events:

  • Add click event:
    events.on('.btn-click-add', 'click', function () { alert('Button click.'); });
  • Add mousedown event:
    events.on('.btn-mousedown-add', 'mousedown', function () { alert('Button mousedown.'); });

Delegate events:

  • Add delegated click event:
    events.on(document, 'click', '.btn-delegate-add', 'click', function () { alert('Delegated button click.'); });

Remove events:

  • Remove click event:
    events.off('.btn-click-remove', 'click');
  • Remove mousedown event:
    events.off('.btn-mousedown-remove', 'mousedown');

Remove delegated events:

  • Remove delegated click event:
    events.off(document, 'click');

Event naming:

  • Add click.myclick event:
    events.on('.btn-myclick', 'click.myclick', function () { alert('Button click named myclick.'); });
  • Remove click.myclick event:
    events.off('.btn-myclick', 'click.myclick');

Trigger events:

  • Trigger click event:
    events.trigger('.btn-click-trigger', 'click');
  • Trigger mousedown event:
    events.trigger('.btn-mousedown-trgigger', 'mousedown');

Check classname:

  • Check element has a class named "new-class":
    events.hasClass('.btn-click-check', 'new-class');

Add/Remove classname:

  • Add a class name "new-class":
    events.addClass('.btn-class-add', 'new-class');
  • Remove a class named "old-class":
    events.removeClass('.btn-class-remove', 'old-class');

Toggle classname:

  • Toggle .hidden classname of "<i>":
    events.toggleClass('.btn-toggle i', 'hidden');

Show/Hide element:

  • Show element "<i>":
    events.show('.btn-show i');
  • Hide element "<i>":
    events.hide('.btn-hide i');

.each() Run for each matched element:

  • Check the console to see the results:

  • Check the console to see the results:
    events.closest('.btn-closest', 'div.parent');
